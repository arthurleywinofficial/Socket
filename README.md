# 🚀 SOCKET - Industrial IIoT Analytics Platform

SOCKET, endüstriyel tesisler için geliştirilmiş, gerçek zamanlı veri izleme, analiz ve simülasyon platformudur.

## ✨ Özellikler
- **Gerçek Zamanlı Dashboard:** Enerji, üretim, lojistik ve güvenlik verileri.
- **Canlı Grafikler:** Sensör verilerinin anlık görselleştirilmesi.
- **Kriz Simülatörü:** Acil durum senaryoları ve ESD testleri.
- **Yapay Zeka Destekli Asistan:** Operasyonel destek.
- **Docker Altyapısı:** Kolay kurulum ve taşınabilirlik.

## 🏗️ Teknoloji Yığını
- **Frontend:** React, Vite, TypeScript, Chart.js, Lucide Icons.
- **Backend:** FastAPI (Python), MQTT, WebSockets.
- **Veritabanları:** PostgreSQL, InfluxDB.
- **Haberleşme:** Eclipse Mosquitto (MQTT Broker).
- **Görselleştirme:** Grafana.

## 🚀 Hızlı Başlangıç
1. `docker-compose up -d`
2. `cd frontend && npm install && npm run dev`
